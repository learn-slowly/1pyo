## 1. 참관인이란?

투개표 참관인은 선거가 공정하게 치러지는지 현장에서 직접 감시하는 사람입니다.

여러분은 **정의당 추천 참관인**으로 활동하게 됩니다. 정의당은 공정한 선거를 위해 가능한 많은 투표소와 개표소에 참관인을 배치하고자 합니다.

### 우리 후보

| 선거 | 기호 | 후보명 | 선거구 |
|------|------|--------|--------|
| (추후 확정) | | | |

> 참관인으로서 후보를 위한 선거운동은 할 수 없지만, 자신이 어떤 정당의 추천으로 참관하는지 아는 것은 기본입니다.

---

## 2. 참관인의 권한과 의무

### 할 수 있는 것

- 투표 및 개표 절차를 가까이서 지켜보기
- 절차 위반이나 부정행위에 대해 이의 제기
- 투개표 과정을 메모로 기록
- 선거관리위원회 위원에게 질문 및 의견 전달

### 해서는 안 되는 것

- 투표소/개표소 안에서 선거운동
- 투표자에게 말을 걸거나 투표에 간섭
- 투표소/개표소 내부 촬영 (허가 없이)
- 정당 상징물(배지, 옷 등) 착용하고 참관
- 투표용지에 손대기
- 음주 상태로 참관

### 위반 시

- 참관인 자격 즉시 박탈
- 공직선거법 위반으로 형사처벌 가능

---

## 3. 투표 참관 (사전투표 / 본투표)

### 투표 절차 흐름

```
유권자 입장 → 본인확인 (신분증) → 투표용지 교부 → 기표소 기표 → 투표함 투입
```

### 감시 포인트

#### 본인확인 단계
- 신분증을 제대로 확인하는가?
- 대리투표 시도는 없는가?

#### 투표용지 교부
- 정해진 투표용지를 정확히 교부하는가?
- 여분의 투표용지가 유출되지 않는가?

#### 기표소
- 기표소 내 비밀투표가 보장되는가?
- 기표소 안에서 촬영하는 유권자는 없는가?
- 2인 이상이 동시에 들어가지 않는가?

#### 투표함
- 투표함 봉인이 훼손되지 않았는가?
- 투표 종료 후 투표함 이동 시 참관인이 동행하는가?

### 사전투표 특이사항
- 사전투표는 통합선거인명부(전자) 사용
- 관외선거인의 경우 회송용 봉투 사용 절차 확인
- 투표지발급기가 정상 작동하는지 확인

---

## 4. 개표 참관

### 개표 절차 흐름

```
투표함 이송 확인 → 개함 → 투표지 분류 → 심사·판정 → 집계 → 결과 공표
```

### 감시 포인트

#### 투표함 확인
- 봉인 상태 확인 — 훼손 흔적이 없는가?
- 투표함 수와 송부서 내용이 일치하는가?

#### 분류 단계
- 투표지 분류기를 사용하는 경우, 오분류가 없는지 확인
- 수작업 분류 시 정확하게 분류하는지 주시

#### 유·무효 판정
- **유효표**: 기표란 안에 기표한 표
- **무효표 사례**:
  - 2개 이상 후보에 기표
  - 기표란 밖에 기표
  - 어느 후보에게 기표한 것인지 식별 불가
  - 다른 표시(낙서, 서명 등)가 있는 경우
  - 정규 투표용지가 아닌 경우
- 유·무효 판정에 이의가 있으면 즉시 이의 제기

#### 집계
- 후보별 득표수 합산이 정확한지 확인
- 총 투표수와 개표된 투표지 수가 일치하는지 확인

## 5. 문제 상황 대응

### 이의 제기 절차

1. **현장에서 구두로** 투표관리관/개표사무원에게 이의 제기
2. 구두로 해결되지 않으면 **서면 이의신청서** 작성
3. 모든 상황은 **시간, 내용, 상대방**을 메모로 기록
4. 즉시 **도당 상황실에 연락** (연락처는 아래 참고)

### 주요 상황별 대응

| 상황 | 대응 |
|------|------|
| 대리투표 의심 | 투표관리관에게 본인확인 재요청 요구 |
| 기표소 내 촬영 | 투표관리관에게 알려 제지 요청 |
| 투표함 봉인 훼손 | 즉시 이의 제기 + 도당 연락 |
| 참관인에 대한 부당한 제한 | 근거 법령 확인 요청 + 기록 + 도당 연락 |
| 개표 시 유·무효 판정 이의 | 해당 투표지에 대해 즉시 이의 제기 |
| 선관위 직원과 마찰 | 감정적 대응 자제, 기록 후 도당에 보고 |

- [ ] 필기구 (펜, 메모장)
- [ ] 휴대폰 + 충전기/보조배터리

- 단정한 평상복
- 정당 로고나 상징색이 드러나는 옷 **착용 금지**
- 특정 후보 관련 표시물 **착용 금지**

### 시간

| 구분 | 시간 |
|------|------|
| 사전투표 | 오전 6시 ~ 오후 6시 (2일간) |
